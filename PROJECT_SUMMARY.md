# joannelsmikle.com — Independent Director Landing Page

Project handoff for GitHub hosting. Client: Dr. Joanne L. Smikle (existing Tenacity IT client, smiklespeaks.com). Goal: a separate, single-purpose site positioning her as an **Independent Director** candidate for board nominating committees and executive search firms. Reference site for tone/simplicity: annmdanner.com. Client is away until July 7; not urgent.

**This repo is public** (required for free GitHub Pages hosting) — that means this file, and everything else in the repo, is publicly readable, not just the rendered site. Don't put anything here you wouldn't want indexed: no personal phone numbers, no non-public contact details, no third parties named without their consent. The current open-item backlog lives in [GitHub Issues](https://github.com/tenacityit/joannelsmikle/issues) rather than as a checklist in this file, partly for that reason and partly because a markdown checklist here can't be checked off from outside a Claude session the way an Issue can.
